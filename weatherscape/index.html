Core Features

당신의 TV를 똑똑한 창문으로

WeatherScape는 실시간 날씨 데이터를 기반으로 영상, 캡션, 음악을 자동으로 선택합니다. 설정할 것은 없습니다. 그냥 켜두세요.

🌤

실시간 날씨 연동

REAL-TIME WEATHER SYNC

OpenWeather API로 현재 위치의 날씨를 실시간 감지합니다. 맑음, 흐림, 비, 눈, 안개 등 10가지 이상의 날씨 상태를 자동 분류합니다.

🎬

자동 영상 전환

AUTO VIDEO TRANSITION

날씨 변화에 따라 적합한 영상이 부드럽게 전환됩니다. 비가 오면 빗소리가 담긴 영상으로, 눈이 오면 설경으로 자연스럽게 바뀝니다.

💬

스마트 캡션

SMART CAPTIONS

현재 온도, 날씨, 위치, 시간대 정보를 화면 하단에 오버레이로 표시합니다. 글래스모피즘 스타일의 세련된 디자인을 적용했습니다.

🎵

BGM 시스템

AMBIENT MUSIC

날씨와 시간대에 맞는 배경 음악을 자동으로 재생합니다. 빗소리, 파도, 재즈, 클래식 등 분위기에 어울리는 사운드스케이프를 제공합니다.

How to Use

3단계로 시작하세요

복잡한 설정 없이 설치 후 바로 즐길 수 있습니다.

1

App Store에서 설치

Apple TV App Store에서 "WeatherScape"를 검색하고 무료로 다운로드하세요.

2

위치 권한 허용

현재 위치의 날씨를 가져오기 위해 위치 접근을 허용해주세요.

3

감상하세요

앱이 자동으로 날씨를 감지하고 최적의 풍경을 보여드립니다. 설정할 것은 없습니다.
